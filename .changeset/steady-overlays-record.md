**Improved:** Recorder overlay controls and aura setup.

The in-game recorder overlay is more compact, follows the selected recording mode, and can start aura setup without leaving gameplay.

- **Recorder overlay:** Uses a single start/stop control, keeps Rewind and Session Recording in sync with the Dashboard, and keeps manual clips available only in Rewind mode.
- **Aura setup:** Adds an aura profile picker and a crop button that opens Aura Manager and immediately starts selecting a new aura region.
- **Overlays:** Aura editing stays visible during crop selection, and recorder, aura, and clip preview overlays can be captured by Windows screenshots and screen recorders.
